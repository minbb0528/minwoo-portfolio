# 김민우 국어 과외 포트폴리오

> 25수능 3등급에서 26수능 1등급으로 상승한 실전 경험을 바탕으로 한 국어 과외 랜딩 페이지

## 📌 프로젝트 소개

중하위권 학생들을 위한 국어 과외 서비스를 소개하는 포트폴리오 웹사이트입니다.
25수능 3등급에서 26수능 1등급으로 도약한 실제 경험을 기반으로, 문학에서 시간을 벌고 독서에서 승부하는 전략적 학습법을 제공합니다.

## 🚀 주요 특징

- **실전 경험 기반**: 25수능 3등급 → 26수능 1등급 달성
- **26수능 문학 전 문항 정답** 달성자의 노하우 전수
- **2단계 로드맵**:
  - STEP 1: 문학 20분 컷 & 전 문항 정답
  - STEP 2: 독서 구조 독해로 1등급 완성
- **중하위권 맞춤형** 밀착 관리 시스템

## 🛠 기술 스택

- **HTML5**: 시맨틱 마크업
- **Tailwind CSS**: 반응형 디자인 및 스타일링
- **AOS (Animate On Scroll)**: 스크롤 애니메이션
- **Font Awesome**: 아이콘
- **Google Fonts**: Noto Serif KR, Pretendard

## 🎨 디자인 특징

- 모바일 우선 반응형 디자인
- 다크 히어로 섹션으로 강력한 첫인상
- Before/After 비교로 성적 향상 시각화
- 그라데이션 효과와 애니메이션으로 현대적인 UI/UX
- 플로팅 상담 버튼으로 전환율 최적화

## 📱 주요 섹션

1. **Hero Section**: 3등급 → 1등급 성과 강조
2. **Problem Analysis**: 타겟 학생들의 학습 문제점 진단
3. **Roadmap**: 2단계 등급 향상 전략
4. **Management**: 중하위권 맞춤형 관리 시스템
5. **CTA**: 카카오톡 오픈채팅 상담 연결
